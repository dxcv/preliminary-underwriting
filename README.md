#### 项目介绍

预核保是在客户投保前，将客户基本资料、意向投保险种、保额、自身身体健康资料等信息通过小程序上传至预核保系统。小程序客户信息录入、附件上传，后台核保人员工作池、核保人员回复、工作量统计、公司管理、公告栏管理。
